# Feature - Onboarding

## Scopo e responsabilità
Il flusso di onboarding accompagna i nuovi utenti nella configurazione del loro profilo e nella spiegazione dei concetti base della piattaforma. È essenziale per la retention iniziale.

## File che compongono questa area
- `src/app/onboarding/` — Route e layout del flusso multi-step
- `src/components/onboarding/` — Step della wizard: benvenuto, selezione interessi, configurazione avatar
