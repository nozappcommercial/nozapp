# Interfaccia Utente

## Scopo e responsabilità
Questa area contiene tutti i componenti UI riutilizzabili, il design system e la gestione delle animazioni globali. Si basa su Radix UI (via Shadcn) e Framer Motion / Animejs per un'esperienza fluida e premium.

## File che compongono questa area
- `src/components/ui/` — Componenti atomici (Button, Input, Card, ecc.)
- `src/components/animations/` — Wrapper e configurazioni per motion design
- `src/components/layout/` — Navbar, Footer, Sidebar, Page transitions
