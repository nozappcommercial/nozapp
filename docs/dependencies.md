# Dipendenze

Di seguito le librerie fondamentali che permettono a NoZapp di funzionare.

## Core Dependencies

*   **`next` (v16.1.6)**: Il framework React per il Web. Fornisce App Router e Server Actions.
*   **`three` (v0.183.2)**: Libreria 3D fondamentale per il rendering WebGL della Sfera Semantica.
*   **`react` & `react-dom` (v18)**: Core engine dei componenti.
*   **`@supabase/supabase-js` & `@supabase/ssr`**: SDK per l'interazione con il database e la gestione dell'autenticazione.
*   **`zod`**: Utilizzato per la validazione degli schemi dati, specialmente nelle API di onboarding.

## UI & Animazioni

*   **`framer-motion`**: Gestisce le transizioni fluide dell'interazione utente e le animazioni del layout.
*   **`animejs`**: Utilizzato per gestire animazioni coordinate, in particolare gli scroll fluidi tra le sezioni della Home.
*   **`lucide-react`**: Set di icone scalabili e pulite.
*   **`tailwindcss-animate`**: Plugin per Tailwind che facilita le animazioni CSS entry/exit.

## Development Dependencies

*   **`typescript`**: Per il controllo statico dei tipi.
*   **`vitest`**: Framework di test per le unit test sulla logica di business e traversata del grafo.
*   **`eslint`**: Per mantenere la qualità del codice e il rispetto delle convenzioni.
